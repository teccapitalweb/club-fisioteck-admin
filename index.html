Guía rápida

Desde este panel puedes gestionar todo el contenido del Club FisioTeck.
1. Agrega/edita cursos grabados, clases en línea y PDFs desde las secciones correspondientes.
2. Gestiona los socios activos y sus suscripciones.
3. Cuando termines tus cambios, presiona "📤 Publicar Cambios" para generar el archivo JSON actualizado.
4. Sube el archivo content.json a tu repositorio de GitHub Pages para que los socios vean los cambios en tiempo real.
